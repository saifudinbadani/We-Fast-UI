Snackbar

To make a snackbar, copy paste the code provided.

This is a sample snackbar

note: I have used a simple background colour and font class to design the snackbar, it works with the javascript for which I have provided the code below.

            
                
                
This is a sample snackbar


                
                const snackbarBtn = document.querySelector('.snackbar-btn');
                const snackbar = document.querySelector('.snackbar')

                function removeSnackbar(){
                    snackbar.classList.remove('show-snackbar')
                }
                function showSnackbar(){
                    snackbar.classList.add('show-snackbar');
                    setTimeout(removeSnackbar() , 3000);
                }

                snackbarBtn.addEventListener('click', showSnackbar)
