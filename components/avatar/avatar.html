Avatars

Round Avatars

To create a round avatar component, add "avatar-round" to the element's class.

Square avatars

To create a square avatar component, add "avatar-square" to the element's class.

Avatars with different sizes

To create avatars with different sizes use "avatar-md", "avatar-sm", "avatar-xs" for medium, small and extra-small size respectively.
